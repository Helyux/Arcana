# Arcana Roadmap

This document outlines the planned features and future improvements for the Arcana browser extension.

## Planned Features

### Visual & Filtering Enhancements
- [ ] **Float Highlighting Colors** — Apply different color intensities or gradients based on the float value (e.g., Factory New vs. Battle-Scarred).
- [ ] **Max Float Filter** — Add the possibility to filter listings by a user-defined maximum float value.
- [ ] **Manual Refresh / Auto-Update** — Make Arcana update the listings occasionally and alert on new matching listings that appear.

### Sorting & Ranking
- [ ] **Float Sorting** — Add the possibility to sort by float instead of the default Steam sorting (by price).
- [ ] **Pattern Ranking (CSV)** — Implement usage of generated `.csv` files to add a rank label for each pattern (e.g., Tier 1, Blue Gem, etc.).
- [ ] **Rank Sorting** — Add the possibility to sort by the assigned pattern rank.

---
Contributions and feedback are welcome! Feel free to open an issue or pull request if you have ideas for additional features.
