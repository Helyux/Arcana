# Arcana Roadmap

This document outlines the planned features and future improvements for the Arcana browser extension.

## Planned Features

### Practical Improvements
- [ ] **Max Float Filter** — Add the possibility to filter listings by a user-defined maximum float value.
- [ ] **Manual Refresh / Auto-Update** — Make Arcana update the listings occasionally and alert on new matching listings that appear.

## Completed Features

### Visual & Filtering Enhancements
- [x] **Float Highlighting Colors** — Color-coded float badges: >0.06 red, <0.01 blue, <0.001 green.
- [x] **In-Page Pattern Filter** — Real-time filtering using pattern templates.
- [x] **Remote Database Integration** — Automatic pattern group identification from `cs2pattern`.

### Sorting & Ranking
- [x] **Float Sorting** — 3-state sorting (Ascending, Descending, Original) by float value.
- [x] **Pattern Ranking** — Manual input ranking system with 1000-entry validation and rank labels.
- [x] **Rank Sorting** — 3-state sorting (Ascending, Descending, Original) by custom pattern rank.
- [x] **Enhanced UI Layout** — Organized filter toolbar with dedicated tool rows for sorting and ranking.
